# MCP Sandbox Layer

Forschungsprototyp zur Bachelorarbeit **Konzeption und Evaluation eines MCP Sandbox Layers zur Trust-Kalibrierung bei LLM-gestützten Änderungen in Low-Code-Plattformen**.

## Struktur

- `backend/` - Backend (Express + MCP, BaselineService, Lowcoder-Adapter, SQLite-Store)
- `frontend/` - Frontend-Skelett (Vue 3 + Quasar)
- `infrastructure/` - Lowcoder Docker-Compose, nginx-Vorlage, Postman-Collection
